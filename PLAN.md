 KẾ HOẠCH THỰC HIỆN ĐỀ TÀI

 1. Tên đề tài
Xây dựng hệ thống quản lý công việc nội bộ cho công ty Unikom

 2. Thông tin sinh viên thực hiện

- Họ và tên: Đoàn Thị Mai Lan
- Vai trò: Sinh viên thực tập năm 3
- Đơn vị thực tập: Công ty Unikom

 3. Giới thiệu đề tài

Hiện nay nhiều công ty sử dụng các công cụ đơn giản như tin nhắn hoặc bảng tổng hợp để quản lý công việc nội bộ, dẫn đến khó theo dõi tiến độ và phân chia nhiệm vụ giữa các nhân viên.

Đề tài “Xây dựng hệ thống quản lý công việc nội bộ cho công ty Unikom” được thực hiện nhằm xây dựng một hệ thống web hỗ trợ quản lý công việc cơ bản trong công ty. Hệ thống giúp quản lý tạo công việc, phân công nhiệm vụ cho nhân viên và theo dõi tiến độ thực hiện công việc một cách dễ dàng hơn.

 4. Mục tiêu đề tài

- Xây dựng giao diện quản lý công việc dễ sử dụng
- Hỗ trợ đăng nhập cho nhân viên và quản lý
- Cho phép quản lý tạo và phân công công việc
- Nhân viên có thể cập nhật trạng thái công việc
- Hỗ trợ theo dõi tiến độ công việc nội bộ

 5. Công nghệ sử dụng

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

 6. Chức năng dự kiến

Đối với nhân viên
- Đăng nhập hệ thống
- Xem danh sách công việc được giao
- Cập nhật trạng thái công việc
- Xem thông báo từ quản lý

 Đối với Admin
- Quản lý tài khoản nhân viên
- Tạo công việc mới
- Phân công công việc cho nhân viên
- Theo dõi tiến độ công việc
- Quản lý danh sách công việc

 7. Kế hoạch thực hiện

 Tuần 1: PLAN + Khảo sát + Phân tích yêu cầu

 Công việc thực hiện
- Xây dựng kế hoạch thực hiện đề tài
- Tìm hiểu mô hình quản lý công việc nội bộ
- Xác định các chức năng chính của hệ thống
- Phân tích yêu cầu cho Admin và Nhân viên
- Xây dựng Use Case cơ bản
- Thiết kế sơ bộ giao diện hệ thống

 Phần em thực hiện 
- Phân tích yêu cầu hệ thống
- Xác định chức năng cần xây dựng
- Thiết kế bố cục giao diện
- Xây dựng nội dung tài liệu

 Phần AI hỗ trợ
- Gợi ý cấu trúc PLAN
- Gợi ý Use Case
- Gợi ý bố cục tài liệu
- Hỗ trợ mô tả chức năng hệ thống

 Tuần 2: Thiết kế Database + UI + API + Lập trình

 Công việc thực hiện
- Thiết kế cơ sở dữ liệu MySQL
- Thiết kế giao diện đăng nhập
- Thiết kế giao diện Admin và Nhân viên
- Xây dựng chức năng đăng nhập/đăng xuất
- Xây dựng chức năng quản lý công việc
- Kết nối PHP với MySQL
- Hiển thị dữ liệu công việc

 Phần em thực hiện 
- Thiết kế UI/UX hệ thống 
- Xây dựng giao diện bằng HTML/CSS
- Thiết kế database
- Code các chức năng chính

 Phần AI hỗ trợ
- Gợi ý cấu trúc database
- Gợi ý giao diện
- Hỗ trợ viết code mẫu
- Hỗ trợ sửa lỗi code cơ bản


 Tuần 3: Kiểm thử + Hoàn thiện + Release

 Công việc thực hiện
- Kiểm tra chức năng đăng nhập
- Kiểm tra chức năng quản lý công việc
- Kiểm tra giao diện hệ thống
- Sửa lỗi phát sinh
- Tối ưu giao diện
- Upload source code lên GitHub
- Hoàn thiện báo cáo và chuẩn bị thuyết trình

 Phần sinh viên thực hiện chính
- Kiểm thử chức năng
- Hoàn thiện giao diện
- Chỉnh sửa và tối ưu hệ thống
- Chuẩn bị demo

 Phần AI hỗ trợ
- Gợi ý test case
- Hỗ trợ tìm lỗi
- Gợi ý cải thiện giao diện
- Hỗ trợ tối ưu hóa đoạn code xử lý

 8. Kết quả mong muốn

Hoàn thành hệ thống quản lý công việc nội bộ cơ bản có thể hoạt động ổn định trên môi trường web, hỗ trợ quản lý công việc hiệu quả hơn cho công ty.
